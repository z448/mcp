# NAME

- mcpq
- Metacpan Query

# SYNOPSIS

- Search for distribution on Metacpan 
- search is case insensitive

!\[alios\](https://raw.githubusercontent.com/z448/alios/master/alios.gif)

# EXAMPLES

> Search for distribution containing 'git' keyword
>
> `mcpq -s git` or `:: -s git`
